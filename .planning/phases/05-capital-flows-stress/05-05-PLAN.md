  <action>
Create COFERCollector using DBnomics API (IMF COFER dataset mirror).

**DBnomics API:**
- Base URL: https://api.db.nomics.world/v22/series/IMF/COFER
- No authentication required
- Returns JSON with quarterly observations

**Series codes (quarterly, world aggregate):**
```python
COFER_SERIES: dict[str, str] = {
    "usd": "Q.W00.RAXGFXARUSD_USD",  # USD reserves
    "eur": "Q.W00.RAXGFXARDEM_USD",  # EUR reserves
    "cny": "Q.W00.RAXGFXARCNY_USD",  # CNY reserves
    "jpy": "Q.W00.RAXGFXARJPY_USD",  # JPY reserves
    "gbp": "Q.W00.RAXGFXARGBP_USD",  # GBP reserves
    "other": "Q.W00.RAXGFXAROT_USD", # Other currencies
}
```

**Key implementation:**
- Use httpx async client for DBnomics API
- Parse JSON response, extract observations
- Calculate currency shares (percentage of total)
- Handle quarterly frequency (Q1=March, Q2=June, etc.)

**Methods to implement:**
- `collect_reserves_by_currency()` -> Reserves in USD millions per currency
- `collect_currency_shares()` -> Percentage share of each currency
- `collect_usd_share()` -> USD share trend over time
- `collect_total_reserves()` -> Total allocated reserves
- `collect()` -> Main entry (currency shares by default)
- `calculate_dedollarization_rate()` -> YoY change in USD share

**Output format:**
- For reserves: timestamp, series_id, source, value, unit (millions_usd)
- For shares: timestamp, series_id, source, value, unit (percent)
- series_id: "cofer_usd", "cofer_eur", "cofer_cny", etc.
- source: "dbnomics_imf"

Register as "cofer" in registry.
