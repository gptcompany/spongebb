### Translation Quality & Confidence

```python
@dataclass
class TranslationResult:
    """Translation output with quality metadata."""
    text: str
    source_lang: str
    model_used: str
    confidence: float  # 0-1, used for sentiment weighting
    is_truncated: bool
```

### Architecture

```
TranslationPipeline
├── __init__(device="auto")  # CUDA if available, else CPU
├── translate(text, source_lang) → TranslationResult
├── translate_batch(texts, source_lang) → list[TranslationResult]
├── detect_language(text) → str  # langdetect
├── _load_model(lang) → pipeline  # Lazy loading
├── _translate_chinese(text) → str  # Qwen3-1.5B special handling
├── _translate_european(text, lang) → str  # Helsinki models
└── _get_device() → int | str

Internal:
├── _models: dict[lang, pipeline]  # Cached models
├── _qwen_model: AutoModelForCausalLM | None  # Loaded if Chinese needed
├── _device: "cuda" | "cpu"
└── _batch_size: int = 16
```

### Key Decisions

- **Helsinki-NLP OPUS-MT for EU langs**: Open-source, fast, trained on EU/UN financial docs
- **Qwen3-1.5B for Chinese**: Better quality on financial terminology than Helsinki zh-en
- **Lazy loading**: Load models on first use to save memory
- **Hybrid GPU/CPU**: Runtime detection at startup (`torch.cuda.is_available()`)
- **Batch processing**: Process up to 16 texts at once
- **Max length**: 512 tokens (truncate if longer)
- **Confidence output**: Each translation includes confidence score for downstream weighting
- **Fallback chain**: If Qwen fails → Helsinki zh-en; if Helsinki fails → return original + flag
