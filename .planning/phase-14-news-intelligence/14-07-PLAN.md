### Architecture

```python
from difflib import SequenceMatcher

class StatementDiffEngine:
    """Word-level diff with sentiment scoring."""

    def __init__(self, sentiment_analyzer: SentimentAnalyzer | None = None):
        self._sentiment = sentiment_analyzer

    def diff(self, old: str, new: str) -> StatementDiff:
        """Generate word-level diff between two statements."""

    def _word_diff(self, old: str, new: str) -> list[DiffOp]:
        """Generate diff operations at word level."""

    def _score_changes(self, diff_ops: list[DiffOp]) -> ChangeScore:
        """Score additions/deletions for hawkish/dovish shift."""

StatementDiff (Pydantic)
├── old_date: date
├── new_date: date
├── operations: list[DiffOp]
├── additions: list[str]
├── deletions: list[str]
├── unchanged_ratio: float (0-1)
├── change_score: ChangeScore
├── html: str  # Rendered HTML

DiffOp (Pydantic)
├── op: Literal["equal", "insert", "delete", "replace"]
├── old_text: str | None
├── new_text: str | None
├── position: int

ChangeScore (Pydantic)
├── direction: Literal["hawkish", "dovish", "neutral"]
├── magnitude: float (-1 to 1)
├── key_changes: list[str]  # Most significant word changes
```

### Key Decisions

- **difflib.SequenceMatcher**: Industry standard, word-level
- **Score aggregation**: Sum of weighted keywords in changes
- **Threshold**: ±0.2 for hawkish/dovish classification
- **Key changes**: Top 5 most impactful word additions/deletions
