Create FXCollector extending BaseCollector[pd.DataFrame].

**Primary source: Yahoo Finance via yfinance**
Tickers:
- DXY: `DX-Y.NYB` (ICE Dollar Index)
- EUR/USD: `EURUSD=X`
- USD/JPY: `USDJPY=X`
- GBP/USD: `GBPUSD=X`
- USD/CHF: `USDCHF=X`
- USD/CAD: `USDCAD=X`
- USD/CNY: `USDCNY=X`
- AUD/USD: `AUDUSD=X`

**Implementation:**
```python
async def _fetch_fx_yahoo(self, symbols: list[str], period: str = "30d") -> pd.DataFrame:
    def _sync():
        import yfinance as yf
        # Single download call for all symbols (avoids rate limiting)
        df = yf.download(symbols, period=period, progress=False)
        return df["Close"]  # Just closing prices
    return await asyncio.to_thread(_sync)
```

**DXY weekend handling:** Apply `ffill()` for weekend NaN gaps (DXY doesn't trade Sunday, FX pairs do).

**Fallback for DXY (Tier 2): FRED**
- Series: `DTWEXBGS` (Nominal Broad USD Index)
- Different calculation but highly correlated

**Standard output format:**
- timestamp, series_id, source, value, unit
- series_id: "DXY" | "EURUSD" | "USDJPY" | etc.
- source: "yahoo" | "fred"
- unit: "index" for DXY, "rate" for pairs

**Methods:**
- `collect_dxy(period="30d")` - DXY only
- `collect_pairs(symbols: list[str], period="30d")` - Specific pairs
- `collect_all(period="30d")` - All FX data

Register: registry.register("fx", FXCollector)
