# Plan 20-02: OPEC Meeting Calendar

## Goal

Integrare calendario meeting OPEC/OPEC+ nel sistema calendar per alert pre-meeting.

## Data Sources

1. **OPEC Official Calendar**: Scrape da pagina ufficiale
2. **Hardcoded known dates**: OPEC+ meeting sono schedulati in anticipo
3. **Fallback**: OPEC-related news come proxy per date meeting

## Known OPEC+ Meeting Pattern

OPEC+ meeting avvengono tipicamente:
- Primo giovedì di ogni mese (JMMC - monitoring)
- Ogni 2-3 mesi per ministerial meeting
- Straordinari su richiesta

### Task 1: Create OPEC calendar module

```python
# src/liquidity/calendar/opec.py

from dataclasses import dataclass
from datetime import datetime, timedelta
from typing import List, Optional

@dataclass
class OPECMeeting:
    """OPEC/OPEC+ meeting information."""
    date: datetime
    meeting_type: str  # "JMMC", "Ministerial", "Extraordinary"
    description: str
    location: Optional[str] = None
    is_confirmed: bool = True

class OPECCalendar:
    """Tracks OPEC/OPEC+ meeting schedule."""

    # Known 2026 meeting dates (to be updated)
    KNOWN_MEETINGS = [
        OPECMeeting(
            date=datetime(2026, 2, 3),
            meeting_type="JMMC",
            description="OPEC+ Joint Ministerial Monitoring Committee",
        ),
        OPECMeeting(
            date=datetime(2026, 3, 3),
            meeting_type="Ministerial",
            description="OPEC+ Ministerial Meeting",
        ),
        # Add more as announced
    ]

    def get_next_meeting(self) -> Optional[OPECMeeting]:
        """Get next scheduled OPEC meeting."""
        now = datetime.now()
        future_meetings = [m for m in self.KNOWN_MEETINGS if m.date > now]
        return min(future_meetings, key=lambda m: m.date) if future_meetings else None

    def get_meetings_in_range(
        self, start: datetime, end: datetime
    ) -> List[OPECMeeting]:
        """Get meetings within date range."""
        return [m for m in self.KNOWN_MEETINGS if start <= m.date <= end]

    def is_meeting_soon(self, days: int = 7) -> bool:
        """Check if OPEC meeting is within N days."""
        next_meeting = self.get_next_meeting()
        if not next_meeting:
            return False
        return (next_meeting.date - datetime.now()).days <= days
```

### Task 2: Integrate with alert system

```python
# In alerts/config.py or new opec_alerts.py:

ALERT_OPEC_MEETING_SOON = AlertConfig(
    name="opec_meeting_soon",
    description="OPEC/OPEC+ meeting in 24 hours",
    threshold=1,  # days
    severity="info",
)

async def check_opec_alerts() -> List[Alert]:
    """Check for upcoming OPEC meetings."""
    calendar = OPECCalendar()
    alerts = []

    next_meeting = calendar.get_next_meeting()
    if next_meeting:
        days_until = (next_meeting.date - datetime.now()).days
        if days_until <= 1:
            alerts.append(Alert(
                type="opec_meeting",
                severity="warning",
                message=f"OPEC {next_meeting.meeting_type} meeting tomorrow",
                data={"meeting": next_meeting},
            ))
        elif days_until <= 7:
            alerts.append(Alert(
                type="opec_meeting",
                severity="info",
                message=f"OPEC {next_meeting.meeting_type} in {days_until} days",
                data={"meeting": next_meeting},
            ))

    return alerts
```
