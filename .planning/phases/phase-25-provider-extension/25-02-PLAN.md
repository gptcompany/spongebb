# Plan 25-02: Fetcher Adapters

## Goal
Implement 3 Fetcher classes that wrap async calculators in the OpenBB TET pipeline pattern.

## Tasks

### T1: NetLiquidity Fetcher
**File:** `src/liquidity/openbb_ext/models/net_liquidity.py`

QueryParams:
- `start_date: Optional[date]` — Start of range (default: 2 years ago)
- `end_date: Optional[date]` — End of range (default: today)

Data fields (from calculator DataFrame columns):
- `date: date` — Observation date
- `net_liquidity: Optional[float]` — Net Liquidity (B USD)
- `walcl: Optional[float]` — Fed Total Assets (B USD)
- `tga: Optional[float]` — Treasury General Account (B USD)
- `rrp: Optional[float]` — Reverse Repo (B USD)

Fetcher class:
- `transform_query()` — Set defaults for start_date/end_date
- `aextract_data()` — Instantiate NetLiquidityCalculator, call `calculate()`, convert DataFrame to list[dict]

### Common Pattern (all 3 fetchers)

**Gate fixes applied:**
1. `AnnotatedResult` imported at module level (not inside method body)
2. All Tier 2 fields explicitly `Optional[float] = None`
3. `transform_query` sets `date` objects (not `.isoformat()` strings)
4. GlobalLiquidity passes `tier` to calculator
5. Calculator imports at module level

```python
from datetime import date, datetime, UTC, timedelta
from typing import Any, Optional

from openbb_core.provider.abstract.annotated_result import AnnotatedResult
from openbb_core.provider.abstract.data import Data
from openbb_core.provider.abstract.fetcher import Fetcher
from openbb_core.provider.abstract.query_params import QueryParams
from pydantic import Field

from liquidity.calculators.xxx import XxxCalculator  # module-level import

class LiquidityXxxQueryParams(QueryParams):
    start_date: Optional[date] = Field(default=None, description="...")
    end_date: Optional[date] = Field(default=None, description="...")

class LiquidityXxxData(Data):
    date: date = Field(description="Observation date")
    # ... fields (all Optional[float] = None) ...

class LiquidityXxxFetcher(Fetcher[LiquidityXxxQueryParams, list[LiquidityXxxData]]):
    @staticmethod
    def transform_query(params: dict[str, Any]) -> LiquidityXxxQueryParams:
        transformed = params.copy()
        if transformed.get("start_date") is None:
            transformed["start_date"] = date.today() - timedelta(days=730)
        if transformed.get("end_date") is None:
            transformed["end_date"] = date.today()
        return LiquidityXxxQueryParams(**transformed)

    @staticmethod
    async def aextract_data(
        query: LiquidityXxxQueryParams,
        credentials: Optional[dict[str, str]],
        **kwargs: Any,
    ) -> list[dict]:
        calc = XxxCalculator()
        start = datetime(query.start_date.year, query.start_date.month, query.start_date.day, tzinfo=UTC)
        end = datetime(query.end_date.year, query.end_date.month, query.end_date.day, tzinfo=UTC)
        df = await calc.calculate(start_date=start, end_date=end)
        df = df.reset_index()
        if "timestamp" in df.columns:
            df = df.rename(columns={"timestamp": "date"})
        return df.to_dict("records")

    @staticmethod
    def transform_data(
        query: LiquidityXxxQueryParams,
        data: list[dict],
        **kwargs: Any,
    ) -> AnnotatedResult[list[LiquidityXxxData]]:
        return AnnotatedResult(result=[LiquidityXxxData.model_validate(d) for d in data])
```

**GlobalLiquidity-specific:** `aextract_data` passes `tier=query.tier` to `calculator.calculate()`.
