# Plan 07-02: Global Liquidity Index

## Objective
Calculate Global Liquidity by aggregating major central bank balance sheets in USD.

## Formula
```
Global Liquidity = Fed Net Liq + ECB_USD + BoJ_USD + PBoC_USD + BoE_USD + SNB_USD + BoC_USD
```

## Coverage Target
- Tier 1 (>85% global flows): Fed, ECB, BoJ, PBoC
- Tier 2 (optional): BoE, SNB, BoC

## Implementation

### File: `src/liquidity/calculators/global_liquidity.py`
```python
class GlobalLiquidityCalculator:
    """Calculate Global Liquidity Index from major central banks."""

    async def calculate(
        self,
        start_date: datetime | None = None,
        end_date: datetime | None = None,
        tier: int = 1,  # 1 = major CBs only, 2 = include minor CBs
    ) -> pd.DataFrame:
        """Calculate Global Liquidity with FX conversion."""

    async def get_current(self) -> GlobalLiquidityResult:
        """Get current Global Liquidity with breakdown."""

    async def _convert_to_usd(
        self,
        value: float,
        currency: str,
        date: datetime,
    ) -> float:
        """Convert CB assets to USD using FX rates."""
```

### Data Types
```python
@dataclass
class GlobalLiquidityResult:
    timestamp: datetime
    total_usd: float  # Total in billions USD
    fed_usd: float
    ecb_usd: float
    boj_usd: float
    pboc_usd: float
    boe_usd: float | None
    snb_usd: float | None
    boc_usd: float | None
    weekly_delta: float
    delta_30d: float
    delta_60d: float
    delta_90d: float
    coverage_pct: float  # % of global monetary base covered
```

### FX Conversion
- EUR/USD for ECB
- USD/JPY for BoJ
- USD/CNY for PBoC
- GBP/USD for BoE
- USD/CHF for SNB
- USD/CAD for BoC
