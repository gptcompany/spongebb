### Model Configuration

```python
TRANSLATION_MODELS = {
    "zh": "Helsinki-NLP/opus-mt-zh-en",  # Chinese → English
    "ja": "Helsinki-NLP/opus-mt-ja-en",  # Japanese → English
    "de": "Helsinki-NLP/opus-mt-de-en",  # German → English
    "fr": "Helsinki-NLP/opus-mt-fr-en",  # French → English
}

# Model sizes: 50-300MB each
# Inference: 400-700 tokens/sec on GPU, 40-80 on CPU
```

### Architecture

```
TranslationPipeline
├── __init__(device="auto")  # CUDA if available, else CPU
├── translate(text, source_lang) → str
├── translate_batch(texts, source_lang) → list[str]
├── detect_language(text) → str  # Optional: langdetect
├── _load_model(lang) → pipeline  # Lazy loading
└── _get_device() → int | str

Internal:
├── _models: dict[lang, pipeline]  # Cached models
├── _device: "cuda" | "cpu"
└── _batch_size: int = 16
```

### Key Decisions

- **Helsinki-NLP OPUS-MT**: Open-source, fast, trained on EU/UN financial docs
- **Lazy loading**: Load models on first use to save memory
- **Hybrid GPU/CPU**: Auto-detect CUDA, fallback to CPU
- **Batch processing**: Process up to 16 texts at once
- **Max length**: 512 tokens (truncate if longer)
