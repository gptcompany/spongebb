  <name>Task 1: Implement CorrelationEngine</name>
  <files>src/liquidity/analyzers/correlation_engine.py</files>
  <action>
    Create CorrelationEngine class:

    **Dataclass: CorrelationResult**
    - timestamp: datetime
    - asset: str (e.g., "BTC", "SPX")
    - liquidity_metric: str (e.g., "net_liquidity", "global_liquidity")
    - corr_30d: float | None (rolling 30-day correlation)
    - corr_90d: float | None (rolling 90-day correlation)
    - corr_ewma: float | None (EWMA with halflife=21)
    - p_value_30d: float | None (significance)
    - p_value_90d: float | None (significance)
    - sample_size: int (number of observations used)

    **Dataclass: CorrelationMatrix**
    - timestamp: datetime
    - assets: list[str]
    - correlations: pd.DataFrame (correlation matrix)
    - p_values: pd.DataFrame (p-value matrix)

    **Class: CorrelationEngine**
    - ASSETS = ['BTC', 'SPX', 'GOLD', 'TLT', 'DXY', 'COPPER', 'HYG'] (per CONTEXT.md)
    - WINDOWS = [30, 90]
    - EWMA_HALFLIFE = 21 (configurable)

    **Methods:**
    - __init__(self, settings: Settings | None = None)
    - calculate_correlations(self, returns: pd.DataFrame, liquidity: pd.Series) -> dict[str, pd.DataFrame]
      - Returns dict with keys '30d', '90d', 'ewma'
      - Each value is DataFrame with asset columns, date index, correlation values
    - calculate_single_correlation(self, asset_returns: pd.Series, liquidity_returns: pd.Series, window: int = 30) -> CorrelationResult
      - For single asset correlation with p-value
    - calculate_correlation_matrix(self, returns: pd.DataFrame) -> CorrelationMatrix
      - Full matrix with p-values using scipy.stats.pearsonr

    **Implementation details (from RESEARCH.md):**
    - Use `pandas.Series.rolling(window).corr()` for fixed windows
    - Use `pandas.Series.ewm(halflife=EWMA_HALFLIFE).corr()` for EWMA
    - Use `scipy.stats.pearsonr()` for p-values (point-in-time only, not rolling)
    - Use `min_periods=window//2` to handle gaps
    - Align timestamps to UTC, forward-fill minor gaps

    **Anti-patterns to avoid:**
    - DON'T use deprecated `pandas.rolling_corr()` - use `Series.rolling().corr()`
    - DON'T correlate raw prices - correlate returns (pct_change)
    - DON'T ignore timezone alignment - BTC trades 24/7, SPX doesn't
    - DON'T skip min_periods - NaN propagation breaks calculations

    **Helper methods:**
    - _fetch_asset_prices(self, symbols: list[str], start_date: datetime) -> pd.DataFrame
      - Use yfinance for price data (same as existing collectors)
    - _calculate_returns(self, prices: pd.DataFrame) -> pd.DataFrame
      - Daily returns via pct_change(), align to common dates

    Use logging (logger = logging.getLogger(__name__))
