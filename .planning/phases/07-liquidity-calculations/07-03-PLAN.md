# Plan 07-03: Stealth QE Score

## Objective
Port the Stealth QE Score from Apps Script v3.4.1 to detect hidden liquidity injections.

## Formula
```
Score = comp1 * 0.4 + comp2 * 0.4 + comp3 * 0.2

Where:
- comp1 (RRP): if rrp_velocity < 0: min(100, |rrp_velocity| / 20 * 100)
- comp2 (TGA): if tga_spending > 0: min(100, tga_spending / 200 * 100)
- comp3 (Fed): if fed_change > 0: min(100, fed_change / 100 * 100)
```

## Config (from Apps Script)
```python
SCORE_CONFIG = {
    "RRP_VELOCITY_MAX": 20,      # % weekly change threshold
    "TGA_SPENDING_MAX": 200,     # $B weekly spending threshold
    "FED_CHANGE_MAX": 100,       # $B weekly change threshold
    "WEIGHT_RRP": 0.40,
    "WEIGHT_TGA": 0.40,
    "WEIGHT_FED": 0.20,
    "MAX_DAILY_CHANGE": 25,      # Smoothing cap
}
```

## Score Interpretation
| Range | Status | Meaning |
|-------|--------|---------|
| 70-100 | VERY_ACTIVE | Major liquidity injection in progress |
| 50-70 | ACTIVE | Stealth QE detected. Bullish |
| 30-50 | MODERATE | Some injection signals. Neutral |
| 10-30 | LOW | Minimal activity |
| 0-10 | MINIMAL | No hidden injection |
