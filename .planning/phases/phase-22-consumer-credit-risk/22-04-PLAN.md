# Plan 22-04: Claude Code Supervision Protocol (Governance + Execution Gates)

**Created**: 2026-02-20  
**Wave**: 4  
**Effort**: S (Small)  
**Priority**: HIGH

## Goal

Documentare formalmente la supervisione operativa di Claude Code per le implementazioni in repo, con checkpoint obbligatori e criteri di escalation.

## Deliverables

1. Protocollo di supervisione Claude Code in formato GSD
2. Checkpoint obbligatori prima/dopo modifiche, test e push
3. Matrice escalation (quando fermarsi e richiedere decisione utente)
4. Tracciabilità minima richiesta per audit/cross-check

## Tasks

### Task 1: Define supervision scope

- Delimitare le attività soggette a supervisione:
  - code edits
  - test runtime
  - commit/push
  - aggiornamenti GSD

### Task 2: Define execution gates

- Gate pre-edit (scope + rischio)
- Gate pre-commit (diff review + impatti)
- Gate pre-push (verifiche minime + note residual risk)
- Gate post-push (hash + file list + stato GSD)

### Task 3: Define escalation policy

- Stop immediato su:
  - modifiche inattese nel worktree
  - incongruenze tra test/documentazione e implementazione
  - richieste distruttive non esplicitamente autorizzate
