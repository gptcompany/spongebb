Aggiungere a FREDCollector:

```python
async def fetch_real_rates(
    self,
    start_date: datetime | None = None,
    end_date: datetime | None = None,
) -> pd.DataFrame:
    """Fetch real rates (TIPS yields) and nominal yields.

    Returns DataFrame with columns:
    - tips_10y: 10-Year TIPS yield
    - tips_5y: 5-Year TIPS yield
    - nominal_10y: 10-Year Treasury yield (DGS10)
    - nominal_5y: 5-Year Treasury yield (DGS5)
    """
    series = ["tips_10y", "tips_5y", "dgs10", "nominal_5y"]
    return await self.fetch_multiple(series, start_date, end_date)
```
