## Tasks

### 1. Create EIA Collector Base

**File:** `src/liquidity/collectors/eia.py`

```python
class EIACollector(BaseCollector):
    """EIA API v2 collector for petroleum data."""

    BASE_URL = "https://api.eia.gov/v2"

    SERIES = {
        "crude_stocks_total": "WCESTUS1",      # US crude stocks
        "crude_production": "WCRFPUS2",         # US production
        "crude_imports": "WCRIMUS2",            # US imports
    }
```

### 2. Implement Async Fetch

- Use `httpx.AsyncClient` for async requests
- API key from settings (`EIA_API_KEY`)
- Rate limiting (1 req/sec to avoid throttling)
- Retry logic via `BaseCollector.fetch_with_retry()`

### 3. Data Normalization

Output format matching existing collectors:
```python
DataFrame(columns=["timestamp", "series_id", "source", "value", "unit"])
```

### 4. Unit Tests

**File:** `tests/unit/collectors/test_eia.py`

- Test series mapping
- Test response parsing

- Mock httpx responses

### 5. Integration Test

**File:** `tests/integration/collectors/test_eia_integration.py`

## API Details

```bash
# Example API call
curl "https://api.eia.gov/v2/petroleum/stoc/wstk/data?api_key=XXX&data[]=value&facets[series][]=WCESTUS1&frequency=weekly"
```

## Notes

- Weekly data released every Wednesday
- Historical data available from 1982
- Max 5,000 rows per request
