### Data Sources

```python
# Primary: Fed official website
FED_FOMC_URL = "https://www.federalreserve.gov/monetarypolicy/fomccalendars.htm"
FED_STATEMENT_PATTERN = "https://www.federalreserve.gov/newsevents/pressreleases/monetary{date}a.htm"

# Fallback: GitHub archives
GITHUB_FOMC_REPO = "https://raw.githubusercontent.com/fomc/statements/main/"

```

### Architecture

```
FOMCStatementScraper(BaseCollector)
├── __init__()
├── collect() → list[FOMCStatement]  # Required by BaseCollector
├── get_latest() → FOMCStatement
├── get_by_date(date) → FOMCStatement
├── get_all_historical() → list[FOMCStatement]
├── _scrape_fed_website(date) → str | None
├── _fetch_github_archive(date) → str | None
└── _parse_statement_html(html) → str

FOMCStatement (Pydantic)
├── date: date
├── meeting_date: date
├── raw_text: str
├── source: Literal["fed", "github"]
├── url: HttpUrl
├── word_count: int
├── fetched_at: datetime
```

### Key Decisions

- **Fed first**: Try official website, fallback to GitHub
- **Historical**: GitHub archives have 1996-present
- **Parsing**: BeautifulSoup for HTML → plain text
- **Caching**: Store fetched statements locally (avoid re-fetching)
- **Format**: Clean text (no HTML tags, normalized whitespace)

### Dependencies

```toml
beautifulsoup4 = ">=4.12.0"
lxml = ">=5.0.0"  # Fast HTML parser
```
