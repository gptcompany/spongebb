# Plan 24-03: Dynamic Date Defaults in Widget Params

## Goal
Add `params` arrays to widget_config for endpoints that accept query parameters, so OpenBB Workspace shows interactive parameter controls with sensible defaults.

## OpenBB Workspace Params Spec (from official docs)
Params are an array of objects:
```json
"params": [
  {
    "paramName": "startDate",   // Maps to query parameter name
    "value": "2024-01-01",      // Default value
    "label": "Start Date",      // Display label
    "type": "date",             // Type: text, number, date
    "show": true,               // Show in widget UI
    "description": "..."        // Tooltip
  }
]
```
