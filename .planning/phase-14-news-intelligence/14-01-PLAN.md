### Feed Configuration

```python
CENTRAL_BANK_FEEDS = {
    "fed_press": "https://www.federalreserve.gov/feeds/press_all.xml",
    "fed_speeches": "https://www.federalreserve.gov/feeds/speeches.xml",
    "ecb_press": "https://www.ecb.europa.eu/rss/press.html",
    "boj_news": "https://www.boj.or.jp/en/rss/whatsnew.xml",
}

# PBoC has no RSS - will be handled via web scraping in later plan
```

### Architecture

```
NewsPoller (async)
├── poll_feed(url) → list[NewsItem]
├── poll_all_feeds() → dict[source, list[NewsItem]]
├── _deduplicate(items) → list[NewsItem]
└── _cache_seen_guids(guids)

NewsItem (Pydantic)
├── source: str (fed, ecb, boj, pboc)
├── title: str
├── link: HttpUrl
├── published: datetime
├── summary: str | None
├── guid: str
├── language: str (en, zh, ja, de, fr)
```

### Key Decisions

- Use `feedparser>=6.0` (RFC-compliant, battle-tested)
- Async HTTP via `httpx` (already in project)
- Poll interval: 30 seconds (configurable)
- Cache last seen GUID per source to avoid duplicates
- Graceful degradation: if feed fails, skip and continue

### Dependencies

```toml
feedparser = ">=6.0"
```
