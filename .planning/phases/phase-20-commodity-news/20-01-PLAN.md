# Plan 20-01: Oil RSS Feeds

## Goal

Aggiungere RSS feeds per notizie oil/energy al news intelligence module.

## Feed Sources

| Source | URL | Tipo | Note |
|--------|-----|------|------|
| OilPrice.com | `https://oilprice.com/rss/main` | RSS | Primario, affidabile |

### Task 3: Create OilNewsPoller class

```python
# src/liquidity/news/oil_feeds.py

class OilNewsPoller(NewsPoller):
    """Polls oil-specific RSS feeds."""

    def __init__(self, feeds: Dict[FeedSource, FeedConfig] = None):
        super().__init__(feeds or OIL_FEEDS)

    async def poll_all(self) -> List[NewsItem]:
        """Poll all oil feeds and return new items."""
        items = []
        for feed in self.feeds.values():
            if feed.enabled:
                new_items = await self._poll_feed(feed)
                items.extend(new_items)
        return items
```
