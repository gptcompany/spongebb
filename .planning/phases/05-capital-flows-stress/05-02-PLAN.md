Create FedCustodyCollector extending existing FredCollector pattern.

**FRED Series (weekly, Wednesday level):**
- WSEFINTL1: Total custody holdings (foreign official + international)
- WMTSECL1: Marketable US Treasury securities (~90% of total)
- WFASECL1: Federal agency debt & MBS (~7% of total)

**Key implementation:**
- Use OpenBB FRED integration (like existing fred.py)
- asyncio.to_thread() for sync OpenBB calls
- Standard output format: timestamp, series_id, source, value, unit

**Methods to implement:**
- `collect_total()` -> Total custody holdings (WSEFINTL1)
- `collect_treasuries()` -> Treasury securities only (WMTSECL1)
- `collect_agencies()` -> Agency debt/MBS (WFASECL1)
- `collect_all()` -> All three series combined
- `collect()` -> Main entry point, returns all by default
- `get_weekly_change()` -> Calculate week-over-week change
- `get_yoy_change()` -> Calculate year-over-year change

**Output format:**
- series_id: "fed_custody_total", "fed_custody_treasuries", "fed_custody_agencies"
- unit: "millions_usd"
- source: "fred"
