<task type="auto">
  <name>Task 1: Create unified overnight rates collector module</name>
  <files>src/liquidity/collectors/overnight_rates.py</files>
  <action>
Create a single module with three collector classes for overnight rates.

**1. ESTRCollector (€STR - Euro Short-Term Rate)**
Primary (Tier 1): estr.dev API (simplest, reliable)
- Endpoint: `https://api.estr.dev/latest`
- Response: `{"date": "2026-01-21", "value": 1.932}`
- Value is in percent (1.932 = 1.932%)

Fallback (Tier 2): FRED
- Series: `ECBESTRVOLWGTTRMDMNRT`
- Via FredCollector

Baseline (Tier 3): BASELINE_VALUE = 2.90 percent (Jan 2026)

**2. SONIACollector (Sterling Overnight Index Average)**
Primary source: FRED only (BoE IADB API unreliable per research)
- Series: `IUDSOIA`
- Via FredCollector with asyncio.to_thread()

Baseline (Tier 2): BASELINE_VALUE = 4.70 percent (Jan 2026)

**3. CORRACollector (Canadian Overnight Repo Rate Average)**
Primary (Tier 1): BoC Valet API (highly reliable, no fallback needed)
- Endpoint: `https://www.bankofcanada.ca/valet/observations/AVG.INTWO/json?recent={n}`
- Response: `{"observations": [{"d": "2026-01-21", "AVG.INTWO": {"v": "2.2500"}}]}`
- Note: Value is STRING, needs `float(obs["AVG.INTWO"]["v"])`

Baseline (Tier 2): BASELINE_VALUE = 3.00 percent (Jan 2026)

**Standard output columns for ALL collectors:**
- timestamp, series_id, source, value, unit
- series_id: "ESTR" | "SONIA" | "CORRA"
- source: "estr_dev" | "fred" | "boc_valet" | "cached_baseline"
- unit: "percent"

**Important:** T+1 publication - €STR and SONIA publish next business day. Label timestamps as "effective_date" (the date the rate applies to).

Register all three: registry.register("estr", ESTRCollector), registry.register("sonia", SONIACollector), registry.register("corra", CORRACollector)
  </action>
  <verify>python -c "from liquidity.collectors.overnight_rates import ESTRCollector, SONIACollector, CORRACollector; print('Imports OK')"</verify>
  <done>Three overnight rate collectors created with appropriate fallback tiers and registered.</done>
