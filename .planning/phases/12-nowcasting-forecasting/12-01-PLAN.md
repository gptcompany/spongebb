# Plan 12-01: Liquidity Nowcast Engine

## Overview

Build a Kalman filter-based nowcasting engine to estimate current Net Liquidity before official Fed releases using high-frequency proxies.

## Goals

- Estimate Net Liquidity 3-5 days before official WALCL release
- Provide confidence intervals for uncertainty quantification
- Handle ragged edge data (mixed daily/weekly frequencies)
- MAPE < 3% vs official releases

## Approach

Use `statsmodels.tsa.statespace.UnobservedComponents` for production-grade state-space modeling with native ragged edge handling.

### State-Space Model

```
State equation:
x_t = [level_t, trend_t]
x_t = F @ x_{t-1} + w_t

Observation equation:
y_t = [TGA_t, RRP_t, SOFR_spread_t, VIX_proxy_t]
y_t = H @ x_t + v_t
```

### Task 3: Implement LiquidityStateSpace Model
**File**: `src/liquidity/nowcasting/kalman/liquidity_state_space.py`

```python
from statsmodels.tsa.statespace.structural import UnobservedComponents
from statsmodels.tsa.statespace.mlemodel import MLEModel
import numpy as np
import pandas as pd
from dataclasses import dataclass
from typing import Optional

@dataclass
class NowcastResult:
    """Result of liquidity nowcast."""
    timestamp: pd.Timestamp
    mean: float                    # Point estimate
    std: float                     # Standard deviation
    ci_lower: float                # 95% CI lower bound
    ci_upper: float                # 95% CI upper bound
    kalman_gain: np.ndarray        # For diagnostics
    innovation: float              # Prediction error

class LiquidityStateSpace:
    """
    Kalman filter nowcaster for Net Liquidity.

    Uses statsmodels UnobservedComponents with local linear trend.
    Handles ragged edge (mixed frequency) via NaN for missing observations.
    """

    def __init__(self, level: str = "local level", trend: str = "local linear trend"):
        self.level = level
        self.trend = trend
        self.model = None
        self.results = None

    def fit(self, endog: pd.Series, **kwargs) -> "LiquidityStateSpace":
        """Fit model on historical data."""
        self.model = UnobservedComponents(
            endog=endog,
            level=self.level,
            trend=self.trend,
            seasonal=None
        )
        self.results = self.model.fit(disp=False, **kwargs)
        return self

    def nowcast(self, steps: int = 1) -> NowcastResult:
        """Generate nowcast for next `steps` periods."""
        if self.results is None:
            raise ValueError("Model not fitted. Call fit() first.")

        forecast = self.results.get_forecast(steps=steps)
        mean = forecast.predicted_mean.iloc[-1]
        ci = forecast.conf_int().iloc[-1]

        return NowcastResult(
            timestamp=pd.Timestamp.now(),
            mean=mean,
            std=forecast.se_mean.iloc[-1],
            ci_lower=ci.iloc[0],
            ci_upper=ci.iloc[1],
            kalman_gain=self.results.filter_results.kalman_gain[:, :, -1],
            innovation=self.results.filter_results.forecasts_error[0, -1]
        )
```

```python
class NowcastEngine:
    """
    Orchestrates nowcasting for multiple liquidity indices.

    Daily workflow:
    1. Fetch HF proxies (TGA, RRP, SOFR)
    2. Validate data quality
    3. Update Kalman filter
    4. Generate nowcast with CI
    5. Store and alert if significant
    """

    def __init__(self, config: dict):
        self.config = config
        self.state_space = LiquidityStateSpace()
        self.validators = []

    async def run_daily_nowcast(self) -> NowcastResult:
        """Execute daily nowcast pipeline."""
        # 1. Fetch latest HF data
        tga = await self._fetch_tga()
        rrp = await self._fetch_rrp()
        sofr = await self._fetch_sofr()

        # 2. Validate
        self._validate_data(tga, rrp, sofr)

        # 3. Update model
        observation = self._combine_proxies(tga, rrp, sofr)

        # 4. Nowcast
        result = self.state_space.nowcast()

        # 5. Check for significant move
        if abs(result.innovation) > 2 * self.config.get("innovation_threshold", 1.0):
            await self._alert_significant_move(result)

        return result
```

### Task 5: Implement Parameter Tuning
**File**: `src/liquidity/nowcasting/kalman/tuning.py`

```python
class KalmanTuner:
    """
    Estimate Q (process noise) and R (measurement noise) matrices.

    Heuristics:
    - R: std(measurement_error)² from TGA/RRP historical errors
    - Q: 5% of returns volatility (liquidity changes slowly)
    """

    @staticmethod
    def estimate_measurement_noise(historical_data: pd.DataFrame) -> np.ndarray:
        """Estimate R from historical measurement errors."""
        # TGA typical error: ±$0.5B
        # RRP typical error: ±$1B
        r_tga = 0.5 ** 2
        r_rrp = 1.0 ** 2
        r_sofr = 0.01 ** 2  # 1bp
        return np.diag([r_tga, r_rrp, r_sofr])

    @staticmethod
    def estimate_process_noise(historical_data: pd.Series) -> np.ndarray:
        """Estimate Q from innovation variance."""
        returns = historical_data.pct_change().dropna()
        q_level = (returns.std() * historical_data.mean()) ** 2 * 0.05
        return np.eye(2) * q_level
```

```python
class NowcastBacktester:
    """
    Walk-forward validation for nowcast accuracy.

    Protocol:
    1. Train on first N days
    2. Nowcast day N+1 using only data up to N
    3. Compare to official release (when available)
    4. Roll forward, repeat
    """

    def run_backtest(self, historical_data: pd.DataFrame,
                     train_window: int = 252) -> pd.DataFrame:
        """Execute pseudo-real-time backtest."""
        results = []

        for test_idx in range(train_window, len(historical_data) - 30):
            # Train only on historical data
            train_data = historical_data.iloc[:test_idx]

            # Fit and nowcast
            model = LiquidityStateSpace()
            model.fit(train_data['net_liquidity'])
            nowcast = model.nowcast()

            # Get official (released 3-5 days later)
            official_idx = test_idx + 5
            if official_idx < len(historical_data):
                official = historical_data.iloc[official_idx]['official']
                error = nowcast.mean - official
                results.append({
                    'date': historical_data.index[test_idx],
                    'nowcast': nowcast.mean,
                    'official': official,
                    'error': error,
                    'pct_error': abs(error / official) * 100
                })

        return pd.DataFrame(results)
```
