# Project Overview
- Name: SpongeBB (`spongebb` in `pyproject.toml`)
- Purpose: Python 3.11+ macro/global liquidity monitoring platform built around OpenBB, FastAPI, Dash, QuestDB, and analytics modules for liquidity, regime detection, risk, news, and backtesting.
- Main package: `src/liquidity`
- Key areas: API (`src/liquidity/api`), dashboard (`src/liquidity/dashboard`), collectors (`src/liquidity/collectors`), nowcasting (`src/liquidity/nowcasting`), validation (`src/liquidity/validation`), risk (`src/liquidity/risk`), alerts (`src/liquidity/alerts`), storage (`src/liquidity/storage`).
