# Plan 07-04: Double-Entry Validation & Coverage Verification

## Objective
Implement data validation to ensure calculation integrity and verify >85% global CB coverage.

## Validation Checks

### 1. Net Liquidity Formula Validation
```python
# Check: Net Liquidity = WALCL - TGA - RRP
calculated = walcl - tga - rrp
reported = net_liquidity
assert abs(calculated - reported) < 0.001  # 0.1% tolerance
```

### 2. RRP Consistency
```python
# RRP should match between Raw_Data and RMP tracker
rrp_from_raw = raw_data["RRPONTSYD"]
rrp_from_tracker = rmp_tracker["rrp_level"]
assert abs(rrp_from_raw - rrp_from_tracker) < 0.01
```

### 3. Global Liquidity Sum
```python
# Fed + ECB + BoJ + PBoC should equal total
components_sum = fed + ecb + boj + pboc
total = global_liquidity
assert abs(components_sum - total) < 0.01
```

### 4. Coverage Verification
```python
# Verify >85% of global monetary flows covered
# Based on IMF estimates of global central bank assets
GLOBAL_CB_ESTIMATE = 35_000  # $35T approximate
coverage = (tier1_total / GLOBAL_CB_ESTIMATE) * 100
assert coverage >= 85.0
```

### 5. Data Freshness
```python
# Data should be no older than expected update frequency
max_age = {
    "WALCL": timedelta(days=7),   # Weekly
    "TGA": timedelta(days=1),      # Daily
    "RRP": timedelta(days=1),      # Daily
    "ECB": timedelta(days=7),      # Weekly
}
```

## Implementation

### File: `src/liquidity/calculators/validation.py`
```python
class LiquidityValidator:
    """Validate liquidity calculations using double-entry checks."""

    async def validate_all(self) -> ValidationResult:
        """Run all validation checks."""

    async def validate_net_liquidity(
        self,
        walcl: float,
        tga: float,
        rrp: float,
        reported_net_liq: float,
    ) -> CheckResult:
        """Validate Net Liquidity formula."""

    async def validate_global_sum(
        self,
        components: dict[str, float],
        reported_total: float,
    ) -> CheckResult:
        """Validate Global Liquidity sum."""

    async def validate_coverage(
        self,
        tier1_total: float,
    ) -> CheckResult:
        """Validate >85% coverage requirement."""

    async def validate_freshness(
        self,
        timestamps: dict[str, datetime],
    ) -> CheckResult:
        """Validate data freshness."""
```

### Data Types
```python
@dataclass
class CheckResult:
    name: str
    passed: bool
    expected: Any
    actual: Any
    tolerance: float | None
    message: str

@dataclass
class ValidationResult:
    timestamp: datetime
    checks: list[CheckResult]
    passed: int
    failed: int
    warnings: int
    coverage_pct: float
    all_passed: bool
```
