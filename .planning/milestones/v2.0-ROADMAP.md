# Milestone v2.0: Advanced Analytics

**Status:** ✅ SHIPPED 2026-02-06
**Phases:** 11-15
**Total Plans:** 21

## Overview

v2.0 adds advanced analytics capabilities to the Global Liquidity Monitor:
- High-frequency data layer (daily TGA, China proxies, stablecoins)
- Nowcasting and forecasting (Kalman filters, HMM, PBoC estimator)
- Risk metrics (VaR, CVaR, Regime VaR)
- News intelligence (RSS, NLP translation, sentiment analysis)
- Backtesting engine (VectorBT, Monte Carlo, regime attribution)

## Milestone Summary

**Key Decisions:**

| Decision | Rationale |
|----------|-----------|
| VectorBT for backtesting | Fast vectorized ops, regime filtering native |
| ALFRED for point-in-time | Avoid look-ahead bias |
| finBERT for sentiment | Pre-trained on financial text |
| akshare for China data | Best maintained package |
| DefiLlama for stablecoins | Free API, comprehensive |

**Issues Resolved:**

- TGA lag reduced from weekly to daily
- China liquidity now proxied via DR007/SHIBOR
- CB communications monitored in real-time
- Look-ahead bias prevented in backtesting
- Strategy validation via Monte Carlo

**Technical Debt Incurred:**

- VectorBT PRO needed for advanced features
- akshare API may change without notice
- Qwen3 requires GPU for fast inference
