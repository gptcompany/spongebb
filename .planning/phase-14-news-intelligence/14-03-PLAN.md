### Architecture

```
SentimentAnalyzer
├── __init__(device="auto")
├── analyze(text) → SentimentResult
├── analyze_batch(texts) → list[SentimentResult]
├── _finbert_score(text) → dict
├── _keyword_score(text) → float
└── _combine_scores(finbert, keywords) → SentimentResult

SentimentResult (Pydantic)
├── sentiment: Literal["positive", "negative", "neutral"]
├── confidence: float (0-1)
├── tone: Literal["hawkish", "dovish", "neutral"]
├── combined_score: float (-1 to 1)
├── finbert_score: float
├── keyword_score: float
```

### Key Decisions

- **FinBERT**: Pre-trained on financial text, 440MB
- **Hybrid scoring**: Dynamic based on translation confidence
  - Native English: 70% FinBERT + 30% keywords
  - Translated text: 60% FinBERT + 40% keywords (boost keywords to compensate)
- **Threshold**: > 0.2 hawkish, < -0.2 dovish, else neutral
- **Max length**: 512 tokens (FinBERT limit)
- **Fallback chain**: FinBERT→fail→keyword-only→fail→return neutral with warning

### Confidence-Weighted Scoring

```python
def _combine_scores(
    self,
    finbert_score: float,
    keyword_score: float,
    translation_confidence: float = 1.0,  # From TranslationResult
) -> SentimentResult:
    """Combine scores with translation confidence weighting.

    For translated text (confidence < 1.0), we:
    1. Reduce trust in FinBERT (trained on English)
    2. Increase weight of keywords (language-agnostic signals)
    """
    # Dynamic weight: more keywords for lower translation confidence
    # Native English: 70/30, Translated Chinese: 60/40
    finbert_weight = 0.7 * translation_confidence
    keyword_weight = 1.0 - finbert_weight

    combined = (finbert_weight * finbert_score) + (keyword_weight * keyword_score)

    return SentimentResult(
        combined_score=combined,
        finbert_weight_used=finbert_weight,
        keyword_weight_used=keyword_weight,
        translation_confidence=translation_confidence,
        # ...
    )
```

### Error Cascading

```python
async def analyze_with_fallback(self, text: str, translation_result: TranslationResult | None = None) -> SentimentResult:
    """Analyze with graceful degradation on failures."""
    try:
        finbert_score = await self._finbert_score(text)
    except Exception as e:
        logger.warning(f"FinBERT failed: {e}, falling back to keyword-only")
        finbert_score = 0.0  # Neutral default

    try:
        keyword_score = self._keyword_score(text)
    except Exception as e:
        logger.warning(f"Keyword scoring failed: {e}, returning neutral")
        return SentimentResult.neutral(reason="analysis_failed")

    confidence = translation_result.confidence if translation_result else 1.0
    return self._combine_scores(finbert_score, keyword_score, confidence)
```
