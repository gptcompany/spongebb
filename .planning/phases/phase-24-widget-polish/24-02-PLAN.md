# Plan 24-02: Column Definitions, Formatters & Renderers

## Goal
Add explicit `columnsDefs` (note: 's' in columnsDefs per OpenBB spec) with `formatterFn` and `renderFn` to all 11 table endpoints missing them (calendar/events already has columnsDefs).

## OpenBB Workspace Supported Values (from official docs)

### formatterFn (allowed values)
- `int` — Format as integer
- `none` — No formatting
- `percent` — Add '%' to number
- `normalized` — Multiply by 100
- `normalizedPercent` — Multiply by 100 and add '%'
- `dateToYear` — Convert date to year

### renderFn (allowed values)
- `greenRed` — Green/red color based on positive/negative
- `titleCase` — Convert text to title case
- `columnColor` — Color column based on rules (with renderFnParams)
- `hoverCard` — Show info on hover
- `cellOnClick` — Trigger action on click
- `showCellChange` — Highlight WebSocket changes
