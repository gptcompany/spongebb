  <action>
Create StressIndicatorCollector for funding market stress indicators.

**FRED Series needed:**
- SOFR: Secured Overnight Financing Rate
- EFFR: Effective Federal Funds Rate (proxy for OIS)
- SOFR1, SOFR99: SOFR 1st and 99th percentiles
- RRPONTSYD: Fed reverse repo operations (daily)
- WALCL: Fed total assets (for normalization)
- DCPF3M: 3-month financial commercial paper rate
- DTB3: 3-month Treasury bill rate

**Stress metrics to calculate:**

1. **SOFR-OIS Spread** (primary funding stress):
   - Calculation: `(SOFR - EFFR) * 100` in basis points
   - Normal: 0-10 bps, Stress: >25 bps

2. **SOFR Distribution Width** (market fragmentation):
   - Calculation: `SOFR99 - SOFR1` in bps
   - Normal: <20 bps, Crisis: >50 bps

3. **Repo Stress Ratio** (Fed intervention level):
   - Calculation: `(RRPONTSYD / WALCL) * 100`
   - Normal: <1%, Elevated: >3%

4. **CP-Treasury Spread** (credit stress):
   - Calculation: `(DCPF3M - DTB3) * 100` in bps
   - Normal: 20-40 bps, Stress: >100 bps

**Methods to implement:**
- `collect_sofr_ois_spread()` -> SOFR-OIS spread time series
- `collect_sofr_distribution()` -> Percentile width
- `collect_repo_stress()` -> RRP/WALCL ratio
- `collect_cp_spread()` -> CP-Treasury spread
- `collect_all()` -> All stress indicators combined
- `collect()` -> Main entry (returns all)
- `get_current_regime()` -> Returns "GREEN", "YELLOW", or "RED" based on thresholds

**Thresholds (for regime classification):**
```python
THRESHOLDS = {
    "sofr_ois": {"green": 10, "yellow": 25},  # bps
    "sofr_width": {"green": 20, "yellow": 50},  # bps
    "repo_stress": {"green": 1, "yellow": 3},  # percent
    "cp_spread": {"green": 40, "yellow": 100},  # bps
}
```

**Output format:**
- series_id: "stress_sofr_ois", "stress_sofr_width", "stress_repo", "stress_cp"
- unit: "basis_points" or "percent"
- source: "calculated" (derived from FRED components)
