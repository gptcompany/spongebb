# Plan 19-02: Breakeven Inflation Calculator

## Goal

Calcolare breakeven inflation (BEI) dalle differenze tra tassi nominali e reali.

## Formulas

```
BEI_10Y = DGS10 - DFII10   (10-year breakeven inflation)
BEI_5Y = DGS5 - DFII5      (5-year breakeven inflation)
5Y5Y_Forward = ((1 + BEI_10Y)^10 / (1 + BEI_5Y)^5)^(1/5) - 1
             ≈ 2 * BEI_10Y - BEI_5Y  (linearized approximation)
```

Il 5Y5Y forward è l'aspettativa di inflazione tra 5 e 10 anni, usato dalla Fed come indicatore chiave.

### Task 1: Create real_rates.py analyzer

```python
# src/liquidity/analyzers/real_rates.py

@dataclass
class RealRatesState:
    """Current state of real rates and inflation expectations."""
    timestamp: datetime
    tips_10y: float
    tips_5y: float
    nominal_10y: float
    nominal_5y: float
    bei_10y: float
    bei_5y: float
    forward_5y5y: float

class RealRatesAnalyzer:
    """Analyze real rates and inflation expectations."""

    def __init__(self, settings: Settings | None = None):
        self.settings = settings or get_settings()
        self._fred = FREDCollector(settings=self.settings)

    async def calculate_breakeven(
        self,
        start_date: datetime | None = None,
        end_date: datetime | None = None,
    ) -> pd.DataFrame:
        """Calculate breakeven inflation series.

        Returns DataFrame with:
        - bei_10y: 10-year breakeven inflation
        - bei_5y: 5-year breakeven inflation
        - forward_5y5y: 5-year, 5-year forward inflation
        """
        df = await self._fred.fetch_real_rates(start_date, end_date)

        df["bei_10y"] = df["nominal_10y"] - df["tips_10y"]
        df["bei_5y"] = df["nominal_5y"] - df["tips_5y"]
        df["forward_5y5y"] = 2 * df["bei_10y"] - df["bei_5y"]

        return df[["bei_10y", "bei_5y", "forward_5y5y"]]

    async def get_current_state(self) -> RealRatesState:
        """Get current real rates and BEI snapshot."""
        df = await self._fred.fetch_real_rates()
        latest = df.dropna().iloc[-1]

        return RealRatesState(
            timestamp=latest.name,
            tips_10y=latest["tips_10y"],
            tips_5y=latest["tips_5y"],
            nominal_10y=latest["nominal_10y"],
            nominal_5y=latest["nominal_5y"],
            bei_10y=latest["nominal_10y"] - latest["tips_10y"],
            bei_5y=latest["nominal_5y"] - latest["tips_5y"],
            forward_5y5y=2 * (latest["nominal_10y"] - latest["tips_10y"])
                         - (latest["nominal_5y"] - latest["tips_5y"]),
        )
```

@pytest.mark.asyncio
async def test_calculate_breakeven():
    analyzer = RealRatesAnalyzer()

    assert not df.empty
    assert "bei_10y" in df.columns

## Cross-Validation

FRED ha già la serie T10YIE (10-Year Breakeven Inflation Rate) che possiamo usare per validare:

```python
async def validate_bei_calculation(self) -> bool:
    """Validate BEI calculation against FRED T10YIE."""
    df = await self._fred.fetch_series("T10YIE")
    bei_calc = await self.calculate_breakeven()

    diff = (bei_calc["bei_10y"] - df["value"]).abs()
    return diff.max() < 0.01  # Max 1bp difference
```
