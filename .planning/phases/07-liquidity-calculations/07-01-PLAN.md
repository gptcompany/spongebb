# Plan 07-01: Net Liquidity Index

## Objective
Implement the Hayes Net Liquidity formula as the core liquidity indicator.

## Formula
```
Net Liquidity = WALCL - TGA - RRP
```

Where:
- **WALCL**: Fed Total Assets (weekly, Wednesday)
- **TGA**: Treasury General Account (daily)
- **RRP**: Reverse Repo Facility (daily)

### Data Types
```python
@dataclass
class NetLiquidityResult:
    timestamp: datetime
    net_liquidity: float  # In billions USD
    walcl: float
    tga: float
    rrp: float
    weekly_delta: float
    monthly_delta: float
    delta_60d: float
    delta_90d: float
    sentiment: str  # BULLISH, NEUTRAL, BEARISH
```

### Sentiment Thresholds
- BULLISH: weekly_delta > $50B
- BEARISH: weekly_delta < -$50B
- NEUTRAL: otherwise
